## はじめに

こんにちは、@norihirosunada です。

p5.js でクリエイティブコーディングをする際の楽しさといえば、「コードだけで試行錯誤して偶然の発見をすること」「目指すイメージに向けて作る」など全く違う方向性があると思います。

ただ、p5.js エディターでは制御性や開発体験としては壁があり、vscodeなどで制作している方もいます。

そこで、第3の選択肢として、Webブラウザ上で p5.js をノードグラフ形式で記述できるエディタ「p5 Node Studio」 の開発を始めました。せっかく作ったので紹介させてください。

## 主な機能とこだわり

1. ノードベースのワークフロー

画面上に配置した「ノード」を線でつなぐことで、処理の流れを構築できます。
Circle ノードに Transform ノードをつなげば位置が変わる、といった具合です。

ノードの出力から線を引っ張ってドラッグ＆ドロップで入力に繋げます。

2. リアルタイムプレビュー & パラメータ調整

ノードのパラメータ（スライダーや数値入力）を操作すると、繋がっているノードに即座に反映されます。

3. Live Code Editing（自作ノード）

![](https://storage.googleapis.com/zenn-user-upload/a1fa6ce7e47e-20251209.png)

ノード同士はp5.Graphics を受け渡して接続されています。ノードの中身をその場でコードとして記述・編集できます。p5.js で書かれているので自分で好きにコーディング可能です。

## 今後の展望（Roadmap）

現在はまだ基本機能のみです。今後は以下のような機能を実装していきたいと考えています。
